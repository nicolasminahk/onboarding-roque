# Capturas reales de la interfaz (opcional)

Si querés reemplazar o acompañar los esquemas de la Sección 02.1 con capturas reales,
guardá acá estos 4 archivos PNG y pedime que los embeba en el documento.

## Lista de tomas

1. `01-ventana-principal.png` — Claude (escritorio o web), ventana principal:
   barra lateral visible (Chats / Proyectos / Artefactos) y el compositor abajo.
   Ideal con el Proyecto de 0800 visible en la barra lateral.
2. `02-conectores.png` — Configuración → Conectores, mostrando Shopify,
   Higgsfield y Magnific conectados.
3. `03-skill-entregable.png` — un chat donde una skill generó un archivo
   (p. ej. una planilla): que se vea el archivo entregable.
4. `04-claude-code.png` — Claude Code en acción (terminal o app de escritorio)
   con un pedido escrito en castellano.

## Recomendaciones

- PNG, ancho mínimo ~1400 px, tema claro u oscuro pero consistente entre las 4.
- Antes de capturar: cerrá o recortá títulos de chats, mails y datos de clientes.
- Las capturas se embeben en base64 dentro del index.html (sigue siendo un solo archivo);
  las 4 juntas suman ~1–2 MB al documento, sin problema para uso interno.
